A React input component combining text and image handling with modern UX. Supports Enter to submit, Shift+Enter for newlines, and drag-and-drop image uploads.

Includes image preview modals, individual deletion, form validation via react-hook-form + Zod, and full accessibility with ARIA labels.

Responsive design optimized for both mobile and desktop.
