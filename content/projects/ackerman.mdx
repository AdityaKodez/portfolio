A minimal, animated landing page template for quick deployment. Features pre-built components optimized for performance with a clean aesthetic.

Built with Next.js 16 + React 19 and styled using Tailwind CSS 4. Uses Radix UI primitives and shadcn/ui for accessible, polished interface elements.

Perfect starting point for portfolios, product showcases, or any project needing a sleek landing page fast.
