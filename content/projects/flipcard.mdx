A React component for creating 3D flip cards with smooth animations. Control the flip state via props and display dual-sided content.

Simple API using FlipCard wrapper with FlipFace children for front and back content. Install through shadcn CLI.

Perfect for interactive showcases, about sections, or any UI needing a reveal effect.
