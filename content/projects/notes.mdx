A modern note-taking app with rich text editing powered by TipTap. Organize notes into folders and switch between dark/light themes.

Changes save automatically as you type. User authentication keeps your notes secure and synced across devices.

Built on Next.js 16 + React 19 with Neon PostgreSQL, Drizzle ORM, and better-auth for authentication.
