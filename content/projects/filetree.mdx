A composable, searchable FileTree component for React built with Radix UI and designed for shadcn/ui.

Features fuzzy search that auto-filters and expands matching folders, keyboard navigation, per-file icons, and animated expand/collapse transitions.

Install via shadcn CLI and compose with modular FileTree, Folder, File, and FileTreeSearch elements.
