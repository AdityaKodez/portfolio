A streamlined Hacker News alternative for sharing and discussing content. Submit links, upvote posts, and engage in threaded discussions.

Features user profiles displaying post and comment histories, with email/password authentication powered by Better Auth.

Built on Next.js 16 + React 19 with Neon's serverless PostgreSQL and Drizzle ORM for data persistence.
