A CLI for creating and managing cryptocurrency wallets on Solana and Ethereum. Generate new wallets or import existing mnemonics.

Supports hierarchical wallet derivation, listing wallets with optional secret visibility, and secure deletion.

Wallet data persists locally at `~/.xine/wallets.json`. Built with Bun runtime.
